## Features

This plugin is a port of the [Atom File Icons](https://github.com/file-icons/atom) for Atom and the icons of
the [Material Theme UI](https://github.com/mallowigi/material-theme-issues) plugin.

### File Icons
Automatically replaces file icons based on their extension, content, or naming patterns:

![File Icons Preview](https://raw.githubusercontent.com/mallowigi/iconGenerator/master/assets/files.png)

### Folder Icons
Contextual folder icons for common patterns like `src`, `docs`, `node_modules`, etc:

![Folder Icons Preview](https://raw.githubusercontent.com/mallowigi/iconGenerator/master/assets/folders.png)

### Additional Features
- **UI Icons**: Enhanced toolbar icons, gutter icons, and other UI elements
- **PSI Icons**: Updated Program Structure Interface icons for classes, methods, variables, and other code elements
- **Customization**: Monochrome filter, custom icon sizes, accent colors, and arrow styles
- **Performance**: Optimized for smooth IDE performance with configurable options
